# NeoFrame Frontend

Frontend oficial do NeoFrame SaaS.

Interface para:

- upload de arquivos
- criação de vídeos
- acompanhar jobs
- login futuro
- dashboard do usuário

## Stack

- React
- Vite
- Tailwind CSS
- Axios
- React Router

---

## Rodando localmente

### Instalar dependências

```bash
npm install
````

### Executar

```bash
npm run dev
```

---

## Build produção

```bash
npm run build
```

---

## Docker

```bash
docker build -t neoframe-frontend:1.0.0 .
docker run -p 3000:80 neoframe-frontend:1.0.0
```
